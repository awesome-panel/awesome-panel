I would love to support and receive your contributions. Thanks.

### Installing

```bash
conda create -n awesome-panel -c conda-forge r-quarto perl
```

### Previewing

```bash
cd docs
quarto preview
```

### Publish

```bash
cd docs
quarto publish
```
